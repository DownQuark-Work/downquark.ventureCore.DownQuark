process

We believe simpler is better. Much better.

To reflect that belief we have structured our current process to include the following technologies and concepts wherever possible.

  • Buildless
    • Minimal Dependencies
      • Libraries (at most) no frameworks
      • lit preferred if using a library
  • CSS
    • CSS before javascript
    • Cube CSS Methodologies
  • Patterns
    • Atomic Design
    • Island Architecture
